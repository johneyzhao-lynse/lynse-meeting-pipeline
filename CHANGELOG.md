# Changelog

## 0.2.0 (2026-05-29)

### New Features

- 用户画像开关控制：新增 `/profile` 命令和 `profile_enabled` 配置项，支持一键切换画像加载
- TUI wizard 画像选择步骤：`/run` 流程中自动检测并提示选择用户画像
- 输出参数行增强：生成总结文件头显示用户画像和动态提示词的启用状态
- README 文档：完整的项目介绍、使用方式、参数说明和项目结构

### Changed

- 修复 TUI 模型命令测试：mock 配置文件替代依赖真实配置
- CLI 输出参数行新增画像和动态提示词状态字段

## 0.1.0 (2026-05-28)

### New Features

- 会议分类流水线：基于关键词的规则引擎 + LLM 路由，自动匹配行业和模板
- 用户画像增强系统：支持角色、行业、会议场景、风格偏好等维度的 prompt 个性化
- 安全内容检测与自动重试：政治、自害、亵渎、隐私四类敏感内容的本地正则扫描
- 交互式 TUI（Rich + Prompt Toolkit）：引导式总结向导、模型切换、配置管理
- 飞轮质量闭环：反馈记录 → 推理模型分析 → prompt 改进建议
- 35+ 结构化总结模板和 14 个行业增强提示词
- 金融调研模板（FinSurvey）
- 评估批次管理（eval-batches）

### Changed

- 引入 `pyproject.toml` 管理项目元数据和版本号
- 修复 TUI 模型命令测试：使用 mock 配置文件替代依赖真实配置
